HTML Tutorial

BODY :- The content which we want to display on the web page we must write inside the body tag.

TAGS in HTML :- The HTML document is created using tags. with names to create a tag.

Types of TAG :-

  • Paired tag :- It contain opening tag and closing tag.
  • Unpaired tag :- It contains only opening tag or self closing tag.
  • Elements in HTML :- It is combination of opening tag, closing tag & content.

Basic TAG in HTML :-

1] Heading Tag :- This tag's create Heading in the HTML document. All heading tag are paired tag & block level elements. They are 6 Types of heading tag describe as below:

  • h1 : 2em
  • h2 : 1.5em
  • h3 : 1.17emp
  • h4 : 1.0em
  • h5 : 0.83em
  • h6 : 0.67em

NOTE :- 1em stands for equal to the current font size. It is a unit measurement.

2] Paragraph tag <p> :- It is used to create a paragraph in webpage & it allows only single space between the character. Line breaks are not allowed.

3] Pre formatted tag <pre> :- Pre formatted tag displays the content as it is written inside the tag. Line breaks & extra spaces are allowed.

4] Div tag <div> :- Id defines the division & section of the document. It acts as a container for block level container & Inline level elements. It is a block level element.

5] Span tag <span> :- It is a inline level element. It acts as a container for inline level elements.

6] Image tag <img> :- It is used to insert the image into your webpage by default it takes two attributes a) src & b) alt.

  • a) :- It is the source that specifies the path of the image document that should be display on a webpage.
  • b) :- Alternative text for the image which is visible on the webpage if the image in the src is failed to fetch the path.
    1. Absolute path :- Absolute path is directly fetching the image from browser.
    2. Relative path :- Relative path is fetching the image from local folder with the help of image/file name.

NOTE :- Atrrbutes are special words used inside opening tag to control elements's behaviour.

Formatting tags in HTML :-

  1. <strong> or <b> tag :- It will display the content in bold texts. If your are using strong tag in place of b tag it will improve the search engine optimization.
  2. <em> or <i> tag :- It will display the content in Italic font style.
  3. <ims> of <u> tag :- It will display the content in underline.
  4. <mark> tag :- It will display content with yellow highlighted bacground.
  5. <q> tag :- It will display the content with in the double quotes.
  6. <del> tag :- It will display the content with strike over the content.
  7. <sup> :- It will display the content just above with respect to the previous content.
  8. <sub> :- It will display the content just below with respect to the previous content.
  9. <small> tag :- It will display the content with smaller text.

Attributes in HTML :-

  • Attributes provide some extra information about the HTML elements.
  • It must be present in the opening tag of the HTML elements.
  • It comes with attribute name = attribute value or only attribute name.
  • There are three types of attributes.
    1. Core Attribute.
    2. Internationalization Attribute.
    3. Elements Specific Attribute.
    1. Core Attribute :-
      There are 4 core attributes in HTML :-
    • id :- The value of id attribute must be unique and it targets only the individual element.
    • class :- We can pass the same value for multiple elements in HTML & the class attribute targets multiple elements.
    • title :- The title attribute create the tooltip. Tooltip is a kind of popup which pops out when the cursor is placed on the targeted element.
    • style :- The style attribute is used to give the style for the HTML element & it is also called as inline CSS, it gives the more priority.
    2. Internationalization Attributes :-
    • lang :- lang attributes defines the language of the content which is present inside the HTML element we have to pass the language code as a value to the lang attributes.
    • dir :- dir attributes defines the direction of the content which is display on the webpage. It takes two value one ltr (Left-to-right text direction) & rtl (Right-to-left text direction). auto is the default value.
    3. Elements Specific Attributes :-
      This Attributes are specific to the perticular element we cannot able to write this attribute for every HTML elements. e.g., Contain two attributes. i = src & ii = alt <img src="" alt="">

List in HTML :- There are 4 types of list in html.

  1. Ordered List :- Ordered List start with <ol>. Each item listed with the tage <li>. We can use 3 attributes in ordered list.
    1. Type :- Default value is 1. It takes input as Number, Alphabets.
    2. Start :- The value of the start is number.
    3. Reversed :- It will reverse the order.
  2. Unordered List :- Unordered List start with <ul>. Each item listed with tags <li>. It contains 1 attribute called type.
    1. Type :- disc, square, circle. Default value is disc.
  3. Description List :- Discription List start with <dl>. Each item listed with tags <dt> & <dd>. By default the description data appears with some tab space.
  4. Nested List :- List inside list.
